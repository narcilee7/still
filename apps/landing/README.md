# Still Landing Page

A static marketing landing page for Still.

## Development

```bash
yarn workspace @still/landing dev
```

Open http://localhost:3000.

## Build

```bash
yarn workspace @still/landing build
```

Outputs to `apps/landing/dist/`.

## Notes

- Built with plain HTML, CSS, and vanilla JavaScript.
- Uses Google Fonts (Inter + Cormorant Garamond).
- All product screenshots are CSS placeholders until real assets are ready.
